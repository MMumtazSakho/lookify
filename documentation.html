Authentication

Use your API key in the header of every request:

Authorization: Bearer YOUR_API_KEY

1. KYC API

Endpoint: POST /api/kyc/register

Description: Submit facial data and ID document to verify identity.

{

2. Face as PIN

Description: Authenticate user via facial recognition.

{

Description: Trigger a payment using face input.
